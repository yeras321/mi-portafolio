Indeal Constructores – UX UI Diseño y Desarrollo Web
📍 Proyecto: Indeal Constructoras – Landing
🎯 Rol: Diseño UX/UI y Desarrollo Web
📆 Duración: Noviembre
🎯 Rol: Desarrolladora Web & UX/UI Directora.
🌐 Sitio web: proyectosindeal.com
Herramientas Utilizadas 🛠️ WordPress | CSS | Figma

📌 Descripción del Proyecto Indeal Constructoras es una plataforma diseñada para conectar constructores de apartamentos con personas interesadas en invertir en bienes raíces. Se desarrolló una landing page estratégica con el objetivo de recolectar información de constructores y darles a conocer sobre la plataforma indeal y optimizar sus ventas atravez de ella.
Responsabilidades en el Proyecto
  • ✅ Estructura y usabilidad: Diseño de una navegación intuitiva para maximizar la conversión.
  • ✅ Desarrollo en WordPress: Implementación completa de la landing page con optimización para captación de leads.
  • ✅ Experiencia centrada en el usuario: Diseño de una interfaz clara y accesible para inversores interesados.

Resultados Clave
✔️ Una landing page optimizada que nos permite llegar a los constructores y puedan conocer la plataforma para llegar a más inversionistas, ofreciendo una experiencia fluida y eficiente para captar clientes potenciales.
